numberOfClauses

var numberOfClauses: Int

Number of not-deleted clauses in the problem. BVE creates and deletes a lot of clauses, which take a lot of memory. We need garbage collector to clean them up, which means we must remove references to deleted clauses from time to time. After eliminating a variable, we check if the number of clauses in the database (including deleted ones) is more than twice the number of not-deleted clauses, which we store here. If it is, we remove all deleted clauses from the db, watchers, and occurrences lists.
