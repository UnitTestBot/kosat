equivalentLiteralSubstitution

private fun equivalentLiteralSubstitution(): SolveResult?

Equivalent Literal Substitution (ELS) is a preprocessing technique that tries to find literals that are equivalent to each other.

Consider clauses (-1, -2), (2, 3) and (-3, 1). In every satisfying assignment, 1 is equal to -2, and -2 is equal to 3. This means that we can substitute, for example, -2 and 3 with 1 in every clause, and the problem will generally remain the same.

This function tries to find such literals and substitute them. To do this, it searches for strongly connected components in the binary implication graph, selects one representative literal from each component, and substitutes all other literals in the component with the representative.
