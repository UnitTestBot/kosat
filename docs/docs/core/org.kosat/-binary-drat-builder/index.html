BinaryDratBuilder

class BinaryDratBuilder(val output: BufferedSink) : AbstractDratBuilder

A DratBuilder that writes the binary proof to a given output.

The binary format is more compact than the text format, but it is not human-readable, therefore, comments are ignored (and not included in the format in the first place).

See Format specification.
